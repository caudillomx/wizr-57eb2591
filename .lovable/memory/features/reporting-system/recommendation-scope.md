## Alcance de Recomendaciones
Las recomendaciones generadas por IA están estrictamente limitadas al ámbito de monitoreo digital y escucha social:
- SÍ: Ajustar keywords, agregar fuentes, configurar alertas, ampliar cobertura, crear dashboards, segmentar análisis, rastrear influenciadores, ajustar frecuencia.
- NO: Comunicados de prensa, estrategia de contenido, campañas de marketing, relaciones públicas, asesoría legal, decisiones operativas.
- Si se detecta riesgo fuera del ámbito digital, se señala como hallazgo sin prescribir acción.

## Lenguaje Cauteloso (Regla Crítica)
- NUNCA afirmaciones absolutas: "No se identificaron...", "No existe evidencia..."
- SIEMPRE hedged: "En la muestra analizada...", "Con base en los datos disponibles...", "De las N menciones recopiladas..."
- Los datos son UNA MUESTRA, no la totalidad del ecosistema mediático.
- Cada hallazgo debe estar respaldado por datos concretos observables.
